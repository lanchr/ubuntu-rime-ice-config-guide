## 一、写在前面

从 Windows 或 macOS 切换到 Ubuntu/Linux 的朋友，大概都有一个共同的痛点：**中文输入法太难用了**。

- 系统自带的 IBus 候选框又丑又卡，打字经常"断片"；
- 曾经的「搜狗输入法 for Linux」在 Wayland 下水土不服，候选框乱跑、甚至无法输入；
- 百度输入法更是早就停止了维护。

作为一个每天要和文字打交道的人怎么能忍？

经过我的反复踩坑和调研，终于找到了一套**在 Ubuntu 24.04 (Wayland 环境) 上完美运行的中文输入方案**：

> **Fcitx5 + Rime (中州韵) + 雾凇拼音词库 + Material 皮肤**

这套方案有多香？这么说吧——**比我之前在 macOS 上用搜狗还要流畅**。

## 三、一条命令搞定

我把整个配置流程封装成了一个**自动化脚本**，你只需要复制下面的命令，粘贴到终端执行即可：

```bash
# 1. 克隆项目
git clone https://github.com/lanchr/ubuntu-rime-ice-config-guide.git
cd ubuntu-rime-ice-config-guide/scripts

# 2. 赋予执行权限
chmod +x setup-rime-ice.sh

# 3. 一键运行（需要输入一次 sudo 密码）
./setup-rime-ice.sh
```

脚本会自动完成以下操作：

- ✅ 检测 Ubuntu 系统环境
- ✅ 安装 Fcitx5 及所有前端依赖
- ✅ 安装 Rime 引擎和 Material 皮肤
- ✅ 拉取雾凇拼音词库并部署
- ✅ 自动切换系统输入法框架
- ✅ 写入配置文件并启动服务

> 💡 **注意**：脚本运行完成后，需要**注销并重新登录**（或重启电脑），输入法才能生效。

## 四、首次使用指南

完成上面的步骤后，按下面的指引进行初始化：

### 1. 切换输入法
按 `Ctrl + Space`，此时应该能切换到 Rime（会有候选框出现）。

### 2. 部署词库
右键点击右上角的键盘图标 → 选择 **「部署(Deploy)」**。

> 第一次部署需要等待 10-30 秒（词库比较庞大），耐心等待即可。

### 3. 选择方案
按 `F4`，在弹出的菜单中选择 **「雾凇拼音」**。

搞定！现在可以开始打字了。

## 六、进阶 customization

### 更换皮肤

默认配置的是「深海蓝」主题，如果你喜欢其他风格，可以修改配置：

```bash
vim ~/.config/fcitx5/conf/classicui.conf
```

可选主题：
- `Material-Color-DeepOcean`（深海蓝）
- `Material-Color-Teal`（水鸭青）
- `Material-Color-Pink`（少女粉）

### 词库同步

如果有多台 Linux 设备，可以把 `~/.local/share/fcitx5/rime/` 目录同步到云盘（OneDrive/坚果云），实现词库多端同步。

### 双拼配置

雾凇拼音默认支持多种双拼方案（小鹤、微软、自然码），同样按 `F4` 切换即可。

## 七、常见问题

**Q: 为什么我按 Ctrl+Space 没反应？**

> A: 确保已经注销并重新登录过。如果还不行，尝试手动执行 `fcitx5 -r` 重启服务。

**Q: 某些应用无法输入中文？**

> A: 大部分现代应用（Chrome、VS Code、Telegram）都没问题。部分老旧 Electron 应用可能需要重启。

**Q: 如何卸载？**

> A: `sudo apt remove fcitx5*` 即可完全卸载。
